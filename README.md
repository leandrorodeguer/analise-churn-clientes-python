# 📊 Análise de Churn de Clientes

## 🎯 Objetivo
Este projeto tem como objetivo analisar os principais fatores que levam ao cancelamento (churn) de clientes, utilizando dados para gerar insights estratégicos e apoiar a tomada de decisão.

---

## 📂 Contexto
Uma empresa com mais de 800 mil clientes identificou um alto número de cancelamentos e precisa entender os motivos para reduzir a evasão.

---

## 🛠️ Ferramentas utilizadas
- Python
- Pandas
- Plotly

---

## 📊 Análise dos Dados

Foram analisados fatores como:
- Tipo de contrato
- Número de ligações ao call center
- Dias de atraso
- Status de cancelamento

---

## 📈 Principais Insights

- Clientes com contrato mensal apresentam maior taxa de cancelamento  
- Clientes com mais interações com o call center têm maior probabilidade de churn  
- Atrasos no pagamento estão fortemente relacionados ao cancelamento  

---

## 💡 Estratégias sugeridas

- Incentivar planos de longo prazo  
- Melhorar a eficiência do atendimento ao cliente  
- Criar políticas para redução de inadimplência
